# Security Policy

## Supported Versions

Unless otherwise stated, only the latest version of this repository is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it privately using one of the following methods:

- GitHub Security Advisories (preferred, if enabled)
- Email the repository owner or maintainers directly

Please include:
- A clear description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact
- Any suggested fixes or mitigations

You can expect:
- An acknowledgment within a reasonable timeframe
- Updates as the issue is reviewed and addressed
- Credit for responsible disclosure, if desired

## Disclosure Policy

We ask that you practice responsible disclosure and avoid publicly sharing details until the issue has been investigated and resolved.

Thank you for helping keep this project and its users secure.
